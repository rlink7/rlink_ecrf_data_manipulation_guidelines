## Running Python code

1. Install Pixi

Linux & macOS

```
curl -fsSL https://pixi.sh/install.sh | bash
```

Windows

```
iwr -useb https://pixi.sh/install.ps1 | iex
```

2. Go to the Python directory, then install the required Python packages (only the first time):

```
cd python
pixi install
```

3. Set up Python configuration:

```
cd python
pixi shell
```
